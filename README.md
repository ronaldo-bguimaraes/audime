# Audime

Gestão detalhada de gastos pessoais

## Instalar dependências

```bash
npm install
```

## Executar web

### To Build
```bash
npm run dev:api
```

### To Dev
```bash
npm run dev:api
```

## Executar api

```bash
npm run dev -w @audime/api
npm run build -w @audime/api
```


## Configurar deploy

### To Build
```bash
npm ci && npm run build:api
```

### To Start API
```bash
npm run start:api
```
